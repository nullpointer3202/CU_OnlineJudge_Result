## 조선대학교 Online-Judge 라이브코딩 점수 산출 코드

조선대학교 OJ 라이브코딩 점수를 크롤링하여 그래프로 통계를 내는 코드입니다.

가상의 크롬 브라우저를 실행하여 온라인 저지 시스템에 접근하는 방식으로 작동하며, 

로그인 과정에서 사용되는 ID/PW 등 모든 사용자 데이터는 해당 프로세스 내에서만 사용되며, 종료와 동시에 사라집니다.

해당 코드의 결과물은 오직 참고용으로만 사용해주시기 바랍니다.


---



시험 결과 텍스트 산출은 현재 oj_selenium_headless.py 파일에서만 가능합니다(26.07.10)

구글 Colaboratory에서 완전히 동작하며 oj_crawler 파일로도 동작 가능합니다(불안정).

2023년에 작성하여 현재 일부만 수정한 미완성 코드이기에 예상치 못한 문제가 발생할 수도 있습니다.

해당 코드 실행 도중 다른 창/기기에서 동일한 ID로 로그인할 경우 자동 로그아웃되어 크롤링이 멈춥니다. 주의해 주세요.
